# Job Portal System (Spring Boot Backend)

This is a backend project built using Spring Boot and MySQL. It provides REST APIs for user authentication and job management.

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- REST API

## Features
- User Registration
- User Login
- Add Job
- View All Jobs

## Tools Used
- IntelliJ IDEA
- Postman
- MySQL Workbench

## Project Structure
- Controller Layer
- Service Layer
- Repository Layer
- Model Layer
